# RCG Forest
Simple green WordPress theme

## About
This is intended to be a simple green theme for WordPress blogs.

This new v2 version of the theme was enhanced with a responsive design, using HTML5 and CSS3, and provides support for recent WordPress features, such as featured images, custom menus, custom background, or custom page templates.

(This theme was originally inspired on Green Grapes theme.  The code base of the current version was adapted from TwentyTwelve theme.)
